# E-commerce Customer Segmentation using Machine Learning

## Project Goal!
This project aims to segment customers based on their purchasing behavior and spending patterns to help e-commerce businesses design targeted marketing strategies.

## Problem Statement
Understanding customer behavior is important for businesses to improve product recommendations, targeted marketing, and customer retention.  
This project applies unsupervised machine learning techniques to identify distinct customer groups.

## Techniques Used
- Data preprocessing and cleaning
- Feature scaling using StandardScaler
- Dimensionality reduction using PCA
- Optimal cluster selection using Elbow Method and Silhouette Score
- Customer segmentation using K-Means clustering
- Data visualization using Seaborn and Matplotlib

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Seaborn
- Matplotlib

## Key Outcome
The model successfully identified multiple customer segments based on income and spending behavior, which can help businesses create personalized marketing strategies.
